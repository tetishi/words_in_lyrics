# WordsInLyrics

WordsInLyricsというサービスは、
スペイン語の単語が覚えられない問題を解決したい
スペイン語学習者 向けの、
言語学習アプリです。
ユーザーは覚えたい単語が使われている曲と歌詞とその訳が表示されるので、その単語の具体的な使い方を知ることができ、
SpanishDictのような辞書兼翻訳アプリとは違って、
単語を使った例文を歌として聴けるため覚えやすくなる要素が備わっている事が特徴です。
